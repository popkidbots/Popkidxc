# 🌟 NEXUS WhatsApp Bot

A modern, fully structured WhatsApp bot built with Node.js, featuring clean UI, automatic command detection, and MEGA session management.

## ✨ Features

- **Modern Command Menu** - Box-style design with auto-detection
- **NEXUS Session System** - Auto-download session from MEGA
- **Modular Architecture** - Easy to expand and maintain
- **Clean UI** - Professional command interface
- **Automatic Commands** - No manual menu updates needed

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/nexus-bot.git
cd nexus-bot

# Install dependencies
npm install

# Create .env file
cp .env.example .env
